About the Key Manager

          

            This simulator implements the ETSI GS QKD 014 REST API. 
            Each Alice–Bob pair shares a pre-generated pool of 256-bit symmetric keys. 
            In production, this module is swapped for a real QKD network — the application interface is identical.
            Key IDs are embedded in X-QKD-KeyID email headers; the key material never leaves the KM.
